# ⚽ Football Tactical Report Engine

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Tests](https://github.com/ScheniD/football-tactical-report-engine/actions/workflows/test.yml/badge.svg)](https://github.com/ScheniD/football-tactical-report-engine/actions)

An advanced football analytics engine that transforms raw event data into professional-grade tactical dossiers. Built for analysts and scouts who need objective, data-driven insights.

## 🚀 Key Features
- **Expected Threat (xT) Analysis:** Quantify player contributions beyond goals and assists.
- **Visual Intelligence:** Generates zonal threat heatmaps, player "pizza" charts, and shot maps.
- **Automated Dossiers:** Compiles complex analytics into a clean, PDF-ready technical report.
- **Data Driven:** Optimized for StatsBomb Open Data integration.

## 🛠 Tech Stack
- **Engine:** Python, Pandas, NumPy
- **Visualization:** Matplotlib, mplsoccer
- **Reporting:** ReportLab
- **Testing:** Pytest
